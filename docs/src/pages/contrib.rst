############
Contributing
############


Setup up the project for development
====================================

.. code-block:: shell

    $ git clone git@github.com:novopl/mappr.git
    $ cd mappr
    $ poetry install
    $ poetry shell


Running tests
.............

.. code-block:: shell

    $ peltak test


Linting
.......

.. code-block:: shell

    $ peltak check


Generating docs
...............

.. code-block:: shell

    $ peltak docs
